## 学到的知识

### PMW3901

用于估计无人机在 X、Y 平面上的运动速度。

可以辅助实现定点悬停。

### VL53L1X

用于测量无人机离地高度。

可以辅助实现定高悬停。

### Git 工作流

Commit：

保存到本地仓库。

Push：

上传到 GitHub 云端。

.gitignore：

忽略编译产生的缓存文件。

---

## 遇到的问题

问题：

IAR 编译后生成大量 .hex、.map、.ninja_log 文件，导致 GitHub Desktop 被淹没。

原因：

这些文件曾经被 Git 跟踪。

解决办法：

配置 .gitignore，并取消 Git 跟踪。
