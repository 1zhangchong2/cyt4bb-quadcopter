# 习惯星球 - 技术架构文档

## 1. 架构设计

```
┌─────────────────────────────────────────┐
│              用户界面层 (UI)              │
│   HTML 页面 + CSS 动画 + DOM 事件处理     │
├─────────────────────────────────────────┤
│              业务逻辑层 (Logic)            │
│   习惯管理 / 打卡逻辑 / 统计计算 / 成就判断 │
├─────────────────────────────────────────┤
│              数据持久层 (Storage)         │
│   LocalStorage (habitPlanet_* 键)        │
└─────────────────────────────────────────┘
```

- **架构风格**：单文件 HTML 应用（Single-File App），无构建工具，零外部依赖
- **状态管理**：内存对象（`habits[]`、`checkins[]`）+ LocalStorage 同步
- **模块划分**：通过 JS 中的 Class/Module 模式组织（HabitStore / CheckinManager / Calendar / Stats / Achievement）

## 2. 技术选型

| 技术 | 用途 |
|------|------|
| HTML5 + CSS3 | 页面结构与样式 |
| Vanilla JavaScript (ES6+) | 业务逻辑 |
| CSS Variables | 主题色管理 |
| CSS Animations + Keyframes | 打卡动画、成就特效 |
| LocalStorage API | 本地数据持久化 |
| Google Fonts | 字体加载（Noto Sans SC / 得意黑） |
| Emoji | 习惯图标（无需图片资源） |
| 内联 SVG | 装饰图形（星球/星星/火箭） |

## 4. 页面结构

```
index.html
├── <head>
│   ├── Google Fonts 引入 (Noto Sans SC / 得意黑)
│   └── <style> 全站 CSS（内联）
│       ├── CSS Variables（颜色/字体/间距）
│       ├── 全局重置样式
│       ├── 动画关键帧（打卡动画/星星雨/徽章弹出）
│       └── 组件样式（卡片/按钮/弹窗/日历）
└── <body>
    ├── #app（主容器）
    │   ├── Header（日期 + 进度圆环）
    │   ├── TabNav（底部导航：今日/日历/统计/设置）
    │   ├── #tab-today（今日打卡面板）
    │   ├── #tab-calendar（日历视图）
    │   ├── #tab-stats（数据统计）
    │   └── #tab-settings（习惯管理与设置）
    ├── #modal-add-habit（添加习惯弹窗）
    ├── #modal-edit-habit（编辑习惯弹窗）
    ├── #toast（轻提示通知）
    └── <script> 全站 JS（内联）
        ├── DataStore（LocalStorage 读写）
        ├── HabitManager（习惯 CRUD）
        ├── CheckinManager（打卡逻辑）
        ├── CalendarView（日历渲染）
        ├── StatsEngine（统计数据计算）
        ├── AchievementEngine（成就判定）
        ├── UIController（界面渲染与事件绑定）
        └── init()（应用初始化入口）
```

## 5. 关键模块说明

### 5.1 DataStore

```javascript
// LocalStorage 读写封装
DataStore.getHabits()        // 读取习惯列表
DataStore.saveHabits(list)   // 保存习惯列表
DataStore.getCheckins()      // 读取打卡记录
DataStore.saveCheckin(record)// 添加单条打卡记录
DataStore.removeCheckin()    // 删除打卡记录（取消打卡）
```

### 5.2 CheckinManager

```javascript
// 打卡核心逻辑
CheckinManager.isCheckedToday(habitId)  // 今日是否已打卡
CheckinManager.checkin(habitId)         // 执行打卡
CheckinManager.undoCheckin(habitId)     // 取消打卡
CheckinManager.getStreak(habitId)      // 计算连续打卡天数
```

### 5.3 AchievementEngine

```javascript
// 成就判定规则
STREAK_BADGES = {
  '7days':  { label: '🌟 连续7天', condition: (streak) => streak >= 7 },
  '30days': { label: '🚀 连续30天', condition: (streak) => streak >= 30 },
  '100days':{ label: '👑 连续100天', condition: (streak) => streak >= 100 },
}
```

## 6. 动画清单

| 动画名称 | 触发时机 | 实现方式 |
|---------|---------|---------|
| 打卡成功粒子爆发 | 习惯打卡后 | CSS @keyframes + JS 动态创建 DOM 粒子 |
| 进度圆环填充 | 页面加载/打卡后 | CSS stroke-dashoffset 动画 |
| 星星雨庆祝 | 获得新成就时 | 多层 @keyframes + absolute 定位粒子 |
| 徽章弹出 | 进入成就页面 | CSS scale(0→1) bounce 缓动 |
| 卡片浮动入场 | Tab 切换时 | CSS transform: translateY + opacity |
| Tab 指示器滑动 | 点击导航项 | CSS transform + transition |

## 7. 响应式策略

- **桌面优先**（设计基于 900px 宽桌面窗口）
- 窗口宽度 < 768px：布局收窄，底部导航保留
- 窗口宽度 < 480px：单列布局，字体略微缩小
